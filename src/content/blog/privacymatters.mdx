PrivacyMatters is an anonymous forum platform built with privacy as the primary design constraint.

> Full write-up coming soon.

## Key features

- No email or registration required
- No IP logging or tracking
- Multi-network support: clearnet, Tor (.onion), and I2P (eepsite)
- File attachments via MinIO object storage
- Real-time updates via Laravel Livewire

## Stack

Built with **Laravel** + **Livewire** on the backend, **PostgreSQL** for storage, and **MinIO** for object storage.
